Contains Synethtic Rmd with 3 Synthetic datasets, Synthetic3.csv is the normal dataset where a candidate is elected or eliminated each time. 
Synthetic_mul.csv contains the data in which 2 candidates were eliminated at the same time twice.
Synthetic_mod.csv is the file in which candidates after first prefernce recieved transfers equally.

The code should run as long as these csv files are included in the environment.
